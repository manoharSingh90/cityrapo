Milestone change log:

MagicSuggest v.2.0.0

  • New home at http://nicolasbize.com/magicsuggest/ to run dynamic examples that gh-pages couldn't handle
  • Now runs with Bootstrap 3 (required)
  • Responsive design
  • No more intrusive loading
